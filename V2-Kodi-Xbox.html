System Configuration

Creating advancedsettings.xml

Create Configuration File:

  1. In Kodi, go to Settings → System → File Manager
  2. Select Profile Directory (special://profile/)
  3. Create new file named advancedsettings.xml
  4. Open with built-in text editor

Insert Configuration Code:

<advancedsettings>
  <!-- CACHE SETTINGS (ESSENTIAL FOR 4K) -->
  <cache>
    <buffermode>1</buffermode>     <!-- Buffer entire video -->
    <memorysize>134217728</memorysize> <!-- 128MB cache -->
    <readfactor>20</readfactor>   <!-- Pre-buffer optimization for HDD -->
  </cache>
  
  <!-- NETWORK OPTIMIZATION -->
  <network>
    <cachemembuffersize>15728640</cachemembuffersize> <!-- 15MB RAM buffer -->
    <curlclienttimeout>30</curlclienttimeout> <!-- Timeout settings -->
  </network>
  
  <!-- XBOX-SPECIFIC TWEAKS -->
  <gui>
    <algorithmdirtyregions>3</algorithmdirtyregions> <!-- Reduce GPU load -->
    <nofliptimeout>0</nofliptimeout>              <!-- Fix GUI stutter -->
  </gui>
  
  <!-- TEXTURE MANAGEMENT -->
  <textures>
    <cachesize>128</cachesize> <!-- 128MB for FENtastic skin -->
  </textures>
</advancedsettings>

Note: After creating this file, you MUST restart Kodi for changes to take effect

Audio/Video Configuration

Access Settings: Kodi Settings → System → Display

  1. Set Resolution to match your display (1080p recommended)
  2. Refresh Rate: Set to 60Hz
  3. Enable Match Refresh Rate on Start/Stop

Audio Settings: Kodi Settings → System → Audio

  1. Output Device: HDMI - Digital Output (Virtual Audio Device)
  2. Output Configuration: Fixed (Set to 48kHz)
  3. Enable Sync playback to display
  4. Set Audio Offset to +100ms if lipsync issues occur

Critical: Disable ALL hardware acceleration options under Settings → Player → Videos

Scraper Testing Procedure

Step-by-Step Testing:

  1. Go to Cocoscrapers → Tools
  2. Select Test Providers
  3. Choose provider to test from list
  4. Wait for test to complete (20-60 seconds)
  5. Check results in Kodi log file:

Interpreting Results:

[SUCCESS] Provider is working properly Safe to enable
[FAILED] 404 Provider website not found Disable permanently
[FAILED] 403 Access forbidden (blocked) Disable temporarily
SSL_ERROR Xbox-specific compatibility issue Disable on Xbox
TIMEOUT Connection too slow Disable or try later

Tip: Always test providers after add-on updates. Providers can stop working without notice.

Widget Management

Refresh vs. Reload:

Action How to Trigger When to Use
Refresh Press VIEW button (≡) Update content without losing position
Reload Context Menu → Reload After Trakt list changes or add-on updates
Rescrape Context Menu → Rescrape When sources are outdated or missing

Widget Optimization:

  1. Set refresh interval to 48 hours (Skin Settings)
  2. Limit widgets to 5-7 per screen
  3. Avoid "auto-updating" widgets
  4. Prefer Trakt lists over dynamic lists

Maintenance & Troubleshooting

Monthly Maintenance Routine

  1. Clear Cache: Fen Light → Tools → Clear Cache
  2. Check Real-Debrid: Verify account status in Fen settings
  3. Clean Library: Settings → Media → Library → Clean Library
  4. Update Add-ons: Check for updates in repository

Quarterly Deep Clean

  1. Delete Thumbnails:
    • File Manager → special://thumbnails
    • Delete all files in folder
  2. Database Cleanup:
    • Navigate to Userdata/Database folder
    • Delete MyVideos*.db files
    • Delete Textures13.db
  3. Reinstall Add-ons:
    • Uninstall and reinstall key add-ons
    • Start with scrapers and players

Log Analysis

Enabling Debug Logging: Settings → System → Logging → Enable debug logging

Finding Log Files:

  • Xbox Path: U:\Users\UserMgr0\AppData\Local\Packages\XBMCFoundation.Kodi_4n2hpmxwrvr6p\LocalCache\Roaming\Kodi\kodi.log
  • Within Kodi: special://logpath/kodi.log

Critical Errors to Monitor:

Win32Networking Network throttling by Xbox OS Reduce cache settings
CAE::Initialize Audio initialization failure Check audio settings
SSL Handshake Failed Scraper compatibility issue Disable problematic provider
MemoryError Xbox out of memory Reduce max threads

Testing & Verification

Post-Configuration Test Protocol

4K Content Test:

  1. Play a 4K REMUX file (60+ GB size)
  2. Check buffering performance:
    • Acceptable: Buffering completes in <2000ms
    • Ideal: Buffering completes in <1000ms
  3. Monitor for frame drops (press O during playback)

Scraper Performance Test:

  1. Find an obscure TV episode (5+ years old)
  2. Initiate playback
  3. Check results:
    • Good: 5-15 sources found
    • Poor: <3 sources found
  4. Note time to first scrape results

Pro Tip: Performance Monitoring

During playback, press O to show performance stats. Watch for:

  • CPU usage should stay below 80%
  • Cache buffer should fill completely during playback
  • No dropped frames during 1080p playback
