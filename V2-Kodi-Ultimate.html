A: Kodi Download/Install/Configuration

1. Download & Install the latest Kodi build (Omega 21.2 as of today) to your Onn 4k TV, Xbox, PC, Fire TV or other target device. https://kodi.tv/download or Microsoft Store for Xbox or Google Play Store for Android or Google TV.
2. Open Kodi and select the ⚙ Settings gear icon in the top left corner of 🏡 Home Screen.
3. This is Kodi Settings
Player Settings
Language Settings
Media Settings
Services Settings
Interface Settings
System Settings

Player Settings

NOTE: "Expert" MUST be selected to view and configure all settings. When you select the Player setting, click "Standard" in bottom left corner until "Expert" is selected.
Player > Videos > Adjust display refresh rate = Off
Player > Videos > Sync playback to display = Enabled
Player > Videos > Display 4:3 videos as = Stretch 16:9
Player > Videos > Render method = Auto Detect (DXVA on XBOX for smooth playback)
Player > Videos > Enable HQ scalers for scaling above = 100%. Try for at least 70% on older/weaker devices
Player > Videos > Allow hardware acceleration = ON (Off XBOX Only)

Player Language Settings

Player > Language > Preferred audio language = English
Player > Language > Prefer default audio streams = ON
Player > Language > Preferred subtitle language = English for always on ( Forced only = embedded by the title)

Media Settings

Media > General > Show parent folder items = Off
Media > Videos > Flatten TV show seasons = If only one season
Media > Videos > Show empty TV shows = ON

Services Settings

Services > General > Device name = Name your device or not 😁
Services > Control > Allow remote control via HTTP = ON (must set username & password 1st)
Services > Control > Allow remote control from applications on this system = ON
Services > Control > Allow remote control from applications on other systems = ON
Services > Control > Username and Password > Choose & Set up Username & Password or not 😁
Services > Caching > Memory Size = 128 MB (Xbox) & Memory Size = 32 MB (Google TV, Fire TV, Onn 4k TV, etc.)
Services > Caching > Read Factor = 20x (Xbox) & Read Factor = 10x (Google TV, Fire TV, Onn 4k TV, etc.)

Interface Settings

Interface > Regional = Default or Configure to your preference
Interface > Screensaver = None

System Settings

System > Audio > Audio output device = HDMI - Digital Output (Virtual Audio Device) or best for your device
System > Audio > Output configuration = Fixed
System > Input > Enable mouse and touch screen support = Off
System > Add-ons > Updates = Never Check for Updates
System > Add-ons > Show notifications = Off
System > Add-ons > Unknown sources = ON
System > Add-ons > Update official add-ons from = Any repositories
✅ Kodi is now installed and configured!

B: Fentastic Skin Install

1. Select 📂 File Manager from the Settings menu.
2. Select Add source
3. Press Enter on <None>
4. Type https://ivarbrandt.github.io/repository.ivarbrandt in the text field.
5. Select OK✔
6. Type "Fentastic" in the bottom media source name text field.
7. Press OK to add the source.
8. Go back to the ⚙ Settings menu and select Add-ons.
9. Install from zip file
10. Select Fentastic
11. Select repository.ivarbrandt-x.x.x.zip to download the Repository.
12. Install from Repository
13. Select ivarbrandt's Repository
14. Select Look and feel
15. Select Skin
16. Select Fentastic > Install (If the skin doesn't switch automatically, Select Fentastic, Select Use)
17. Select YES to switch to this skin.
18. Select YES to Keep Skin
19. Select Yes/OK to the Pop-up.
✅ Fentastic is now installed!

G: Fentastic Skin Configuraton (Widgets)

TIP: For a video tutorial on Widget Configuration & more, visit: https://odysee.com/@kodi_stuff:b/Create-a-kodi-Build-Fentastic-video-2:d Widgets start at about 5 min.

Movies Configuration

1. Main menu items > Movies > select Set main menu path > Fen Light >> Use Fen Light as path > Rename to Movies > select OK for Movies
2. Main menu items > Movies > select Set widgets > Widget 1: > Fen Light >> Movies >> Trending >> Use Trending as path > select Ok > Choose widget display type > Poster
3. Widget 2: > Fen Light >> Movies >> Genres >> Use Genres as path > select OK > Category > Make Genres a stacked widget? Yes > Landscape

TV Shows Configuration

1. Main menu items > TV Shows> select Set main menu path > Fen Light >> TV Shows >> Use TV Shows as path > select Enter > select OK for TV Shows
2. Main menu items > TV Shows> select Set widgets > Widget 1: > Fen Light >> TV Shows >> Trending >> Use Trending as path > select OK > Choose widget display type > Poster
3. Widget 2: > Fen Light >> TV Shows >> Networks >> Use Networks as path > select OK > Category > Make Networks a stacked widget? Yes > Landscape
TIP: You can play around and add more widgets to any of the categories you find in Fen Light!

The Crew & DaddyLive Installation

1. Go to the ⚙ Settings menu
2. Select 📂 File Manager
3. Select Add source
4. Press Enter on <None>
5. Type https://team-crew.github.io in the text field.
6. Select OK ✔
7. Type "The Crew" in the bottom media source name text field.
8. Press OK to add the source.
9. GO BACK TO THE SETTINGS MENU and select Add-ons (You may have to hit the back button multiple times)
10. Install from zip file
11. Select The Crew
12. Select repository.thecrew.x.x.zip
13. Install from repository
14. Select The Crew Repo
15. Select Video Add-ons
16. Select "The Crew"
17. Select Install & OK
18. SCROLL DOWN ⬇ & SELECT "DaddyLive" from that same list.
19. Select Install & OK
20. Select DaddyLive > Configure > the 18+ Adult Channels > Password "lol"
21. Enter the code & Select OK ✔
22. Go back to the Add-ons list & Select The Crew
23. Select Configure > Navigation tab > 18+ Adult Playground > Password "lol"
24. Select OK ✔
25. Back out of The Crew completely
NOTE: Upon Kodi Restart, new categories will have been added to DaddyLive Live TV & The Crew.
✅ The Crew & DaddyLive Sports is now installed.
