A: Kodi Download/Install/Configuration

1. Download & Install the latest Kodi build (Omega 21.2 as of today) to your Onn 4k TV, Xbox, PC, Fire TV or other target device. https://kodi.tv/download or Microsoft Store for Xbox or Google Play Store for Android or Google TV.
2. Open Kodi and select the ⚙ Settings gear icon in the top left corner of 🏡 Home Screen.
3. This is Kodi Settings

🎬 Player Settings

NOTE: "Expert" MUST be selected to view and configure all settings. When you select the Player setting, click "Standard" in bottom left corner until "Expert" is selected.
Media > General > Adjust display refresh rate = Off
Player > Videos > Display 4:3 videos as = Stretch 16:9
Player > Videos > Render method = Auto Detect or (DXVA on XBOX for smooth playback)
Player > Videos > Enable HQ scalers for scaling above = 100%. Try for at least 70% on older/weaker devices
Player > Videos > Allow hardware acceleration = ON (Off XBOX Only)

🗣️ Player Language Settings

Player > Language > Preferred audio language = English
Player > Language > Prefer default audio streams = ON
Player > Language > Preferred subtitle language = English for always on or (Forced only = embedded by the title)

💾 Media Settings

Media > General > Show parent folder items = Off
Media > Videos > Flatten TV show seasons = If only one season
Media > Videos > Show empty TV shows = ON

🌐 Services Settings

Services > General > Device name = Name your device or not 😁
Services > Control > Allow remote control via HTTP = ON (must set username & password 1st)
Services > Control > Allow remote control from applications on this system = ON
Services > Control > Allow remote control from applications on other systems = ON
Services > Control > Username and Password > Choose & Set up Username & Password used NOWHERE else or not 😁
Services > Caching > Memory Size = 128 MB (Xbox)
Services > Caching > Read Factor = 20x (Xbox)

🖥️ Interface Settings

Interface > Regional = Default or Configure to your preference
Interface > Screensaver = None

⚙️ System Settings

System > Audio > Audio output device = HDMI - Digital Output (Virtual Audio Device) or Default
System > Audio > Output configuration = Fixed
System > Input > Enable mouse and touch screen support = Off
System > Add-ons > Updates = Never Check for Updates
System > Add-ons > Show notifications = Off
System > Add-ons > Unknown sources = ON
System > Add-ons > Update official add-ons from = Any repositories
Kodi is now installed and configured!

Real-Debrid WebDAV Setup

1. Get RD WebDAV Creds: Real-Debrid → Security → Devices & Apps → Note username/password.
2. In Kodi: Videos → Files → Add videos → Browse → Add network location
3. Configure:
Protocol: WebDAV (HTTPS) ⚠️ Not WinLib/SMB!
Server: dav.real-debrid.com
Port: 443
Username/Password: Your WebDAV credentials
Remote Path: Blank or /Downloads/
4. Name source: RealDebrid → Set content to "None"
CRITICAL: When adding the source, double-check protocol is set to "WebDAV (HTTPS)" - cursor misplacement is common!
