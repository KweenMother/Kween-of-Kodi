üîë E: Fen Light Accounts Tab Settings

NOTE: These steps are numbered in the order they appear in Fen Light Settings Accounts Tab

#1 Authorize Trakt

1. Settings > Add-ons > My add-ons > Video Add-ons
2. Select Fen Light > Open > Toolsüõ†
3. Select Settings
4. Accounts > TRAKT > Select Authorize
5. Note the Authorization code
6. Using a web browser visit trakt.tv/activate and enter the authorization code (You must be logged into your Trakt account)
7. Select Yes to authorize Fen Light with Trakt.
‚úÖ Trakt is now integrated with Fen Light!

#2 Authorize OMDB

1. Accounts > OMDB > API KEY
2. Enter the Key code emailed from omdbapi.com.
3. Select OK
‚úÖ OMDB is now enabled in Fen Light!

#3 Enable CocoScrapers

1. Accounts > External Scrapers > Enable = ON
2. Accounts > External Scrapers > Choose External Scrapers Module > Select CocoScrapers Module
3. Select OK
‚úÖ CocoScrapers is now enabled in Fen Light!

#4 Authorize Real-Debrid

1. Accounts tab > Real Debrid > Select Authorize
2. Note the Authorization code
3. Using a web browser visit real-debrid.com MY Devices, Add a new Device & enter the authorization code (You must be logged in to your Real-Debrid account).
4. Type "Fen Light" and press Set that name.
5. Back in Kodi, you will see a success message. Press OK to continue.
6. Enable SEARCH CLOUD STORAGE and its categories
7. Priority set to 1
‚úÖ Real-Debrid is now integrated with Fen Light!

Authorize All Debrid

1. Accounts tab > All Debrid > Select Authorize
2. Note the Authorization code
3. Using a web browser visit alldebrid.com pin code and enter the authorization code (You must be logged in to your All Debrid account).
4. Back in Kodi, you will see a success message. Press OK to continue.
5. Enable SEARCH CLOUD STORAGE & its categories
6. Priority set to 2 or above
7. Scroll üëàüèæ Left & üÜô up to the General Tab
‚úÖ All Debrid is now integrated with Fen Light!
